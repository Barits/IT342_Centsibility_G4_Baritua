## Contributing to Documentation

When adding new documentation:

1. **Use Markdown format** (.md files)
2. **Include a table of contents** for longer documents
3. **Add diagrams** where helpful (use Mermaid, PlantUML, or images)
4. **Keep it updated** - documentation should reflect current state
5. **Link between documents** - create a web of knowledge
6. **Update this README** when adding new documents
